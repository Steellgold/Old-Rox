# Interserver Channel

## Setup a channel

Define a room where messages will be received:

```c
!interserver channel [mention]
```

{% hint style="info" %}
 Replace \[mention\] to be a channel mention
{% endhint %}

![](../../.gitbook/assets/image%20%2817%29.png)
